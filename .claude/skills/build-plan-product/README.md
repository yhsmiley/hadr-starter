# `build-plan-product` agent skill

Start from a high-level idea, and systematically turn that into a detailed product description.

Here's a lightning talk, on what this is about:

[![](https://i.ytimg.com/vi/yZNkGqRLrPw/maxresdefault.jpg)](https://www.youtube.com/watch?v=yZNkGqRLrPw&list=PLjyCRcs63y804L55-eIJtrNxMBFbbroL8&)

## Setup

```shell
npx skills add -g bguiz/build-agent-skills --skill build-plan-product
```

## Usage

Within your gen AI agent harness, invoke the skill, e.g.

```text
Help me turn my idea into a product design
```
